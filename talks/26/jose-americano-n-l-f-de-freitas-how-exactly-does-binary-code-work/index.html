Now imagine trying to do it using only the numbers 1 and 0.Ahora imagina hacerlo usando solo los números 1 y 0. Every time you use the Internet to watch a movie,Cada vez que usamos Internet para ver una película, listen to music,escuchar música, or check directions,o comprobar direcciones, that’s exactly what your device is doing,eso es exactamente lo que hace tu dispositivo, using the language of binary code.al usar el lenguaje del código binario.

Computers use binary because it's a reliable way of storing data.Los ordenadores usan el binario por ser una forma fiable de almacenar datos. For example, a computer's main memory is made of transistorsPor ejemplo, la memoria principal del ordenador está hecha de transistores that switch between either high or low voltage levels,que cambia de niveles altos a bajos de voltaje, such as 5 volts and 0 volts.como 5 voltios y 0 voltios. Voltages sometimes oscillate, but since there are only two options,Los voltajes a veces oscilan, pero dado que solo hay dos opciones, a value of 1 volt would still be read as "low."un valor de 1 voltio sería aún considerado "bajo". That reading is done by the computer’s processor,El procesador del ordenador lleva a cabo la lectura which uses the transistors’ states to control other computer devicesmediante los estados del transistor para controlar los dispositivos informáticos according to software instructions.siguiendo las instrucciones del software.

The genius of this system is that a given binary sequenceLo brillante de este sistema es que la secuencia binaria dada doesn't have a pre-determined meaning on its own.no tiene un significado predeterminado por sí sola. Instead, each type of data is encoded in binaryAl contrario, cada tipo de datos está codificado en binario according to a separate set of rules.siguiendo un conjunto independiente de normas.

Let’s take numbers.Tomemos los números. In normal decimal notation, En una notación decimal normal, each digit is multiplied by 10 raised to the value of its position,cada dígito se multiplica por 10 elevado a su posición, starting from zero on the right.empezando por el 0 a su derecha. So 84 in decimal form is 4x10⁰ + 8x10¹.Así 84 en forma decimal es 4x10⁰ + 8x10¹ Binary number notation works similarly,La notación de un número binario funciona de forma similar, but with each position based on 2 raised to some power.pero cada posición basada en 2 elevado a una potencia.

Binary code is also used for far more complex types of data.El código binario también se usa en formas más complejas de datos. Each frame of this video, for instance,Cada fotograma de este video, por ejemplo, is made of hundreds of thousands of pixels.está hecho de cientos de millones de píxeles. In color images,En las imágenes a color, every pixel is represented by three binary sequencescada pixel está representado por tres secuencias binarias that correspond to the primary colors.que corresponden a los colores primarios. Each sequence encodes a numberCada secuencia codifica un número that determines the intensity of that particular color.que determina la intensidad de ese color en particular. Then, a video driver program transmits this informationEntonces, un driver de vídeo transmite esta información to the millions of liquid crystals in your screena los millones de cristales líquidos de tu pantalla to make all the different hues you see now.para crear los diferentes tonos que ahora ves.

The sound in this video is also stored in binary,El sonido del vídeo también está almacenado en binario, with the help of a technique called pulse code modulation.con la ayuda de una técnica llamada modulación por impulsos codificados. Continuous sound waves are digitizedLas ondas de sonido continuas se digitalizan by taking "snapshots" of their amplitudes every few milliseconds.tomando "instantáneas" de sus amplitudes cada pocos milisegundos. These are recorded as numbers in the form of binary strings,Estos se registran como número en forma de cadenas binarias, with as many as 44,000 for every second of sound.con hasta 44 000 por cada segundo de sonido. When they’re read by your computer’s audio software,Cuando el software de audio del ordenador las lee the numbers determine how quickly the coils in your speakers should vibratelos números determinan cómo vibran los altavoces to create sounds of different frequencies.para crear sonidos en diferentes frecuencias.

All of this requires billions and billions of bits.Todo ello requiere miles de millones de bits. But that amount can be reduced through clever compression formats.Pero puede reducirse esa cantidad con formatos de compresión inteligente. For example, if a picture has 30 adjacent pixels of green space,Por ejemplo, si una imagen tiene 30 píxeles adyacentes en un espacio verde, they can be recorded as "30 green" instead of coding each pixel separately -se puede grabar "30 verde" en lugar de separar cada pixel por separado, a process known as run-length encoding.un proceso conocido como codificación RLE. These compressed formats are themselves written in binary code.Estos formatos comprimidos también están escritos en código binario.

So is binary the end-all-be-all of computing?¿Así que el binario es el punto final de la informática? Not necessarily.No necesariamente. But so far, none of these has providedPero hasta ahora, ninguno ha dado as much physical stability for data storage and transmission.tanta estabilidad física para el almacenamiento y transmisión de datos. So for now, everything you see,Por ahora, todo lo que ves, hear,oyes, and read through your screeny lees en tu pantalla comes to you as the result of a simple "true" or "false" choice,es el resultado de una elección simple de "verdadero" o "falso", made billions of times over.hecha miles de millones de veces.
