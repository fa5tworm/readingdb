Computers use binary because it's a reliable way of storing data.أجهزة الكمبيوتر تستخدم النظام الثنائي لأنه وسيلة يمكن الاعتماد عليها لتخزين البيانات. The genius of this system is that a given binary sequenceعبقرية هذا النظام تكمن في أن تسلسل معين من الاعداد الثنائية doesn't have a pre-determined meaning on its own.ليس له دلالة محددة مسبقًا بحد ذاته. So 84 in decimal form is 4x10⁰ + 8x10¹.إذن الرقم 84 في الصيغة العشرية هو 4x10⁰ + 8x10¹. Binary code is also used for far more complex types of data.نظام الترميز الثنائي يستخدم أيضًا لأنواع أكثر تعقيدًا من البيانات. The sound in this video is also stored in binary,الصوت في هذا الفيديو يتم تخزينه أيضًا بصيغة رمز ثنائي، with the help of a technique called pulse code modulation.بمساعدة تقنية تسمى تمثيل الشيفرة النبضية.
